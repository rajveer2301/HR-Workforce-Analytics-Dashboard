# HR Workforce Analytics & Employee Retention Dashboard

## Project Overview

This project analyzes workforce performance, employee attrition, retention risk, promotions, and leadership impact using Power BI, SQL, and Excel.

The dashboard was developed using a dataset containing 50,000 employee records and provides actionable insights for HR decision-making.

## Tools & Technologies

* Power BI
* SQL (MySQL)
* Microsoft Excel
* DAX
* Git & GitHub

## Dataset Information

* Total Employees: 50,000
* Data Domain: Human Resources (HR Analytics)
* Key Areas:

  * Workforce Overview
  * Attrition Analysis
  * Retention Risk
  * Employee Performance
  * Promotions & Leadership Impact

## Dashboard Pages

### 1. Executive Workforce Overview

* Total Employees
* Average Salary
* Average Performance Rating
* Average Job Satisfaction
* Employees by Department
* Gender Distribution
* Employment Status Distribution

### 2. Employee Attrition & Retention Analysis

* Attrition Count
* Attrition Rate
* High-Risk Employees
* Reasons for Employee Exit
* Retention Risk Distribution
* Attrition by Department

### 3. Performance, Promotion & Leadership Analysis

* Promotion Trends
* Performance Rating Analysis
* Training Effectiveness
* Leadership Impact
* Internal Promotion Analysis

## Key Insights

* Workforce Size: 50,000 Employees
* Average Salary: 75.09K
* Average Performance Rating: 3.00
* Average Job Satisfaction: 2.99
* Attrition Rate: 33.26%
